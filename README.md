# Goldentouch Barbershop Website

Static one-page website for Goldentouch Barbershop in Hamilton.

## What is included

- `index.html` — complete site with embedded CSS and JavaScript
- `assets/gallery/` — Booksy gallery/logo images copied into the project
- `package.json` — local static server command
- `CODEX_NOTES.md` — source notes and verification summary
- Items for sale section with `Red Styling Dust for Men Powder Wax`

## Run

```bash
npm run start
```

Then open:

```text
http://localhost:8000
```
